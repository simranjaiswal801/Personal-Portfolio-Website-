# Personal Portfolio Website

## Description
This is a responsive personal portfolio website created using HTML5 and CSS3. It showcases my profile, skills, projects, resume, and contact information.

## Features
- About Me Section
- Skills Section
- Projects Section
- Resume Download
- Contact Form
- GitHub and LinkedIn Links
- Responsive Design

## Technologies Used
- HTML5
- CSS3
